食品成本掃描網站
這是一個專為高齡者和小朋友設計的食品掃描網站，通過掃描成本表來檢視食品的安全性。目的是幫助用戶簡單快速地判斷食品是否含有不適合高齡者或小朋友的成分，並根據以下三個關鍵標準提供判斷：

是否含有糖
是否含有添加物
是否含有反式脂肪
功能介紹
本網站透過掃描食品的成本表，提供即時回應，並回答以下三個問題：

有沒有糖：判斷產品中是否含有過多糖分，對於高齡者和小朋友來說，過多糖分會影響健康。

有沒有添加物：一些食品可能添加了人工色素、香精或防腐劑，這些添加物對健康有潛在風險，尤其是對兒童和老年人。

反式脂肪：反式脂肪是對心血管有害的脂肪，特別是對高齡者和小朋友的健康至關重要。

Marketake (市集/市場)
隨著健康意識的提升，食品標籤越來越受到消費者的關注。此掃描平台可以作為市場上一個有力的工具，協助消費者選擇更符合健康需求的食品。特別是：

高齡者市場：隨著年齡的增長，老年人對健康的關注也日益增加。本網站讓高齡者能夠迅速判斷哪些食品對其健康可能帶來風險，避免糖分、添加物或反式脂肪的過度攝取。

兒童市場：家長可以依照孩子的健康需求，選擇適合他們的食品，避免危害兒童健康的成分，這對於快速選擇食品非常方便。

健康飲食市場：隨著更多人開始關注營養和食品安全，這個工具可為健康飲食愛好者提供快速的食品成分判斷，幫助他們做出更有意識的選擇。

使用方式
掃描食品成本表：只需要掃描食品的成本表（可上傳圖片或直接輸入內容），系統將自動檢查並返回是否含有糖、添加物或反式脂肪的回應。

即時回應：根據掃描的資料，網站會提供清晰的判斷，讓您知道該產品是否適合高齡者或小朋友食用。

技術架構
前端：使用 React 進行頁面展示，確保介面簡單且直觀。
後端：使用 Python 進行食品成本表的解析，並依照設定規則檢查食品成分。
圖像識別：採用 OCR 技術進行成本表圖片中的文字識別。
